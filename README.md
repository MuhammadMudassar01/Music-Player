# 🎵 Music Player

A modern and responsive Music Player built with **HTML, CSS, and JavaScript**. This project was developed as part of my frontend development journey to strengthen my understanding of DOM manipulation, event handling, and interactive user interface design.

## ✨ Project Overview

The Music Player provides a simple and intuitive interface that allows users to control audio playback while enjoying a clean and responsive design. Building this project helped me gain practical experience in creating dynamic web applications using vanilla JavaScript.

## 🚀 Features

- ▶️ Play and Pause functionality
- ⏭️ Next and Previous track controls
- 🎵 Dynamic audio management
- 📱 Responsive design for different screen sizes
- 🎨 Clean and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📚 What I Learned

While building this project, I improved my understanding of:

- DOM Manipulation
- JavaScript Event Handling
- Audio API Integration
- Responsive Web Design
- Frontend Project Structure
